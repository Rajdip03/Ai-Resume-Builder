# AI Resume Builder 
Description: 
A full-stack MERN application to that automates professional resume creation using AI content generation,automated background removal, and live URL sharing for recruiters. 
<br>
# Features:
* Auth: Secure User Login/Signup.
* resume creation using AI content generation.
* automated background removal.
* live URL sharing for recruiters
* Dashboard:
* Resume management (Create/Edit/Delete).
* Editor: Personal info, Experience, Education, and Skills.
* AI Tools: Content optimization and image processing.
* Export: Real-time preview and shareable live links.
<br>
# Tech Stack
-Fronted:React,Tailwind CSS
-Backend: Express,Node
-DataBase: MongoDB
-AI: Google Gemini AI
